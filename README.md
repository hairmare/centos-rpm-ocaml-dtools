# CentOS 7 RPM Specfile for ocaml-dtools

This repository contains the specfile for ocaml-dtools which is part of the [RaBe liquidsoap distribution](https://build.opensuse.org/project/show/home:radiorabe:liquidsoap).

## Usage

```bash
yum install http://download.opensuse.org/repositories/home:/radiorabe:/liquidsoap/CentOS_7/home:radiorabe:liquidsoap.repo
yum install ocaml-dtools
```
